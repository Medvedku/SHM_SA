Since the path to the bucket mirror has changed, you must update the local destination path in the downloader.

### Updating the Download Path
1. Open the downloader and click on **Settings**.
2. In the popup window, locate the **Local Folder** field.
3. Click **Browse...** and select the new directory location.

{{< figure src="/SHM_SA/A_git/B01_PathChange.png" link="/SHM_SA/A_git/B01_PathChange.png" width="400" >}}

4. Save your changes by clicking the **Save** button. A success notification should appear to confirm the configuration has been updated.
